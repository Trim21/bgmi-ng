================================
Welcome to BGmi's documentation!
================================

``BGmi`` v3 是 ``BGmi`` 的新版本

在v3版本，番剧更新以 ``Subscription`` 为单位。
一个 ``Subscription`` 可以包含来自多个数据源的多个 ``series``。
这跟之前版本的 ``BGmi`` 不同，所以也不能由旧版本升级而来。


所有的数据源和下载工具都是插件化的，可以通过 ``pip`` 安装新的数据源。
